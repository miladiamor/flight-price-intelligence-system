# Cheap Flight Scraper

This folder contains the scraping and data preparation pipeline for the Smart Flight Price Advisor project.

## Purpose

The goal of this module is to collect flight data from Google Flights, clean it, enrich it, and prepare it for machine learning.

## workflow
Route list
    ↓
Google Flights scraper
    ↓
Raw flight data
    ↓
Cleaning
    ↓
Feature engineering
    ↓
Route and airport enrichment
    ↓
Final training dataset
